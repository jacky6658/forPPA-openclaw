# 02-還原與重裝

本資料夾收錄 **備份還原與重新安裝** 相關文檔。

## 文檔一覽

| 檔案 | 說明 |
|------|------|
| **Clawdbot-從備份還原與重裝指南.md** | 從您自己的備份還原、依官方實作與安全文檔重裝、重裝後對應原防護層級（allowFrom、Control UI、憑證、技能、Warn：3） |
| **Clawdbot-還原與舊機清理完整指南.md** | 還原流程與舊機清理的完整步驟與注意事項 |

## 使用時機

- 已有一份完整備份（例如 `clawdbot-full-backup-YYYYMMDD` 或 `openclaw_backup`），要在本機或新機還原。  
- 要卸載後重新安裝，且希望保留記憶與知識再還原。  
- 需對照「安全與安裝健檢報告」在還原後恢復相同防護（DM 權限、Control UI 僅本機、憑證權限、技能安全）。

還原前請確認備份來源路徑與 `01-安裝與入門` 中的本機還原指令、安裝位置一致。
