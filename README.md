# 🦞 OpenClaw（AI 龍蝦）完整學習指南

歡迎來到 OpenClaw（原名 ClawdBot）的完整學習資源庫！

本指南專為**非工程師背景的學員**設計，用白話文帶您從零開始，一步步建立自己的 24/7 AI 助理。無論您是想在本機安裝、處理錯誤、還是進階擴充功能，這裡都有完整且易懂的教學。

## 🎯 什麼是 OpenClaw？

OpenClaw 是一個**開源的 AI 助理專案**，讓您可以：
- 📱 透過 **Telegram**（或其他通訊軟體）與 AI 對話
- 🤖 連接多種 **AI 模型**（GPT、Claude、Gemini 等）
- 🧠 擁有**長期記憶**，AI 會記得您之前說過的話
- 🔧 安裝**技能（Skill）** 擴充功能（做簡報、拍照、管理行事曆等）
- 🏠 完全在**您的電腦**上運行，資料隱私有保障

簡單來說，就是打造一個**屬於您自己的 AI 助手**，24 小時待命、記得所有對話、能做各種任務！

## 🔒 安全提醒

### ⚠️ 重要：保護您的隱私

1. **不要分享您的設定檔**  
   `~/.openclaw/openclaw.json` 或 `~/.clawdbot/clawdbot.json` 含有您的 API keys 和 tokens，**不要**分享給任何人或上傳到網路。

2. **不要上傳備份資料到公開平台**  
   `05-備份資料/` 資料夾含有您的對話記錄、憑證等敏感資料，**絕對不要**上傳到 GitHub、Google Drive 等公開平台。

3. **妥善保管 Telegram Bot Token**  
   Bot Token 就像您家鑰匙，任何人拿到都能控制您的 Bot。如果不小心洩漏，立刻到 @BotFather 重新產生新 Token。
