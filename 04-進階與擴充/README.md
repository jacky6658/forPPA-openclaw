## 使用時機

- **安全**：要規範 Agent 在本機執行指令與檔案操作時，依「AI Agent 安全操作電腦規則」設定白名單與禁止行為。  
- **技能**：要安裝或管理技能時，先看「Clawd Skill 管理教學」；要做 PPT 技能時看「PPT Skill 多模板教學」。  
- **整合**：要接 Google Workspace 或本地語音轉文字時，依對應 MCP 與語音設置文檔操作。  
- **週邊**：相機快照、提醒事項、自動化與記憶管理，依上述週邊安裝與 Bot 指南操作。
