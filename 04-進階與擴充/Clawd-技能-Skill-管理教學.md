## 📖 閱讀前必讀：這是什麼、何時用、有必要嗎？

### 這是什麼？
本文件是 **Clawd 技能（Skill）的安裝與管理 SOP**，教您如何：
- 查看目前已安裝的技能
- 從 ClawdHub 安裝新技能
- 啟用／停用／移除技能
- 管理技能的目錄結構

### 何時需要看這份文件？
| 情境 | 是否需要 | 說明 |
|------|---------|------|
| **想安裝新技能擴充 Bot 功能** | ✅ 需要 | 依「三、從 ClawdHub 安裝新技能」操作 |
| **想知道目前安裝了哪些技能** | ✅ 需要 | 依「二、查看目前已安裝技能」操作 |
| **想停用或移除某個技能** | ✅ 需要 | 依後續章節操作 |
| **不知道技能是什麼、存放在哪裡** | ✅ 需要 | 依「一、Clawd 技能是什麼？」了解 |
| **只是要用 Bot 對話、不需要擴充功能** | ❌ 不需要 | 基本功能已足夠，無須安裝額外技能 |
| **第一次安裝 OpenClaw** | ❌ 暫不需要 | 先完成基本安裝與設定，之後需要擴充功能時再回來看 |

- **Clawdbot-PPT-技能-多模板教學.md**：具體的 PPT 技能安裝與使用教學

### 技能是什麼？
**技能（Skill）** = Bot 的擴充功能模組，例如：
- **PPT 生成技能**：讓 Bot 能產生簡報
- **相機拍照技能**：讓 Bot 能控制相機拍照
- **提醒事項技能**：讓 Bot 能管理蘋果提醒事項

所有技能都存放在 `~/clawd/skills/` 目錄。

### 必要嗎？
- **如果您只需要基本對話、記憶、模型切換功能** → 不必要，OpenClaw 預設功能已足夠
- **如果您想讓 Bot 做更多事情（生成 PPT、拍照、操作 Google Workspace 等）** → 必要，這是安裝擴充功能的唯一指南

## 🧠 一、Clawd 技能是什麼？

Clawd 的「技能（Skill）」就是擴充功能模組，  
所有技能都存放在：

```
~/clawd/skills
```

每一個資料夾 = 一個技能。

---

## 📁 二、查看目前已安裝技能

在終端機執行：

```bash
ls ~/clawd/skills
```

---

## ⬇️ 三、從 ClawdHub 安裝新技能

```bash
npx clawdhub install 技能名稱
npx clawdhub sync
```

---

## ▶️ 四、啟動 Clawd

```bash
npx clawd
```

---

## 📂 五、打開技能資料夾

```bash
open ~/clawd/skills
```

---

## 🗑 六、移除技能

```bash
rm -rf ~/clawd/skills/技能名稱
npx clawdhub sync
```

---

## ❗ 常見錯誤

```bash
npx clawdhub login
```
